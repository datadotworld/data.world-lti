You've enabled Canvas, now what?

With Canvas, you can incorporate data.world into your courses with Canvas's world-class LMS.

Create an Organization on data.world.

To make sure you can invite all your students to upload and manage data, you'll first want to create an organization for your class on data.world:

  1. Click here to enter details and create your organization
  2. Invite members
    You can easily invite members via link instead of manually entering each email address
    3. Sharing this link will prompt users to create an account if necessary and request an invite to the org. You will need to accept any requests to join your org.

Install the data.world app in Canvas.

Once you've created your classroom org on data.world, you can install data.world's LTI into your Canvas course:

  1. Navigate to the desired course in Canvas and access the course settings menu
  2. Within the course settings menu, access the course "Apps" menu
  3. Select View App Configurations, then click the Add App button, (+ App), from within the course apps submenu. This will cause an application configuration modal to pop up
  4. Fill out the configuration settings modal, using the link from the Config Link Generator form for the Configuration URL.
